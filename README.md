# Epidemic Graph Simulator

An interactive web-based simulator that demonstrates how an epidemic can spread through a network of connected individuals. The project uses graph data structures to represent people as nodes and connections as edges, then visualizes how infection moves through the network over time.

## Features

- Interactive graph/network visualization
- Epidemic spread simulation
- Multiple node states such as susceptible, infected, and recovered
- Real-time visual updates during the simulation
- Charts and statistics for tracking outbreak progress
- Modular JavaScript files for graph logic, simulation logic, visualization, and charts

## Technologies Used

- HTML
- CSS
- JavaScript
- Graph data structures
- Browser-based visualization

## How to Run

1. Download or clone the repository.
2. Open `index.html` in a web browser.
3. Start the simulator and adjust the available simulation settings.
4. Observe how the infection spreads through the graph.

No installation is required because the project runs directly in the browser.

## What This Project Demonstrates

This project connects data structures and real-world simulation by showing how graph theory can model the spread of disease through a population. It demonstrates:

- Nodes and edges in graphs
- Network-based propagation
- Simulation modeling
- Front-end visualization
- Modular JavaScript organization
